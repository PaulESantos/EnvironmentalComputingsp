Subconjuntos de datos

A menudo queremos subconjuntar nuestros datos, ya sea para examinar filas o columnas particulares de nuestro conjunto de datos, o para extraer observaciones con propiedades especficas. Podemos realizar subconjuntos en la etapa de exploracin de datos para investigar si nuestra variable de respuesta difiere en su relacin con una variable predictora especfica entre categoras (por ejemplo, para buscar una interaccin). Tambin podemos querer extraer elementos de nuestro conjunto de datos para anlisis separados.

En este ejercicio, cargaremos un conjunto de datos donde se muestrearon murcilagos en un bosque de regeneracin en el sureste de Australia que ha sido adelgazado para reducir la densidad de rboles.

El conjunto de datos incluye mediciones del nmero total de llamadas de murcilagos (Actividad) y el nmero de llamadas de murcilagos que indican comportamiento de forrajeo (Forrajeo) registradas durante una noche en 47 sitios y un total de 173 noches de muestreo. Se han recopilado variables relacionadas con el adelgazamiento del bosque y se enumeran como columnas que terminan en adelgazado. Como el bosque estudiado estaba en una llanura de inundacin, tambin se recopilaron covariables relacionadas con la disponibilidad de agua y se enumeran en columnas que terminan en agua.

Primero, descarga el conjunto de datos, Bats_data.csv, y crgalo en R.

Bats <- read.csv(file = "Bats_data.csv", header = T, stringsAsFactors = F)

Fundamentos basicos para subconjuntos en R

Verifica la estructura despus de haber cargado los datos con la funcin str.

Ahora que tenemos una idea del conjunto de datos con el que estamos trabajando, podemos usar los operadores [] y $ para seleccionar filas y columnas en un marco de datos. Es bastante simple: colocamos el nombre del marco de datos del cual queremos seleccionar antes de los corchetes y dentro de los corchetes colocamos una coma. Los nmeros a la izquierda de la coma representan las filas que deseamos seleccionar y los nmeros a la derecha de la coma representan las columnas que deseamos seleccionar.

Por ejemplo, si quisiramos seleccionar solo datos de las primeras tres filas y todas las columnas, usaramos:

Bats[1:3, ]

Si quisiramos solo las ltimas cuatro columnas y todas las filas, usaramos:

Bats[, 7:10]

En combinacin, esto seleccionara datos solo de las primeras tres filas de las ltimas cuatro columnas:

Bats[1:3, 7:10]

Si los nmeros de fila y columna que deseas no forman una secuencia, podemos usar la funcin >c() para concatenar ndices de fila o columna. Por ejemplo, agreguemos la sexta fila y la segunda columna a nuestra seleccin anterior:

Bats[c(6, 1:3), c(2, 7:10)]

En lugar de averiguar qu variable est en qu columna numerada, a menudo es ms fcil usar los nombres de las variables en un marco de datos y seleccionar la variable usando el operador $. Por ejemplo, para elegir solo la variable Site:

Bats$Site

Subconjunto de filas en dplyr

El paquete dplyr tiene muchas funciones convenientes para el subconjunto de datos que pueden ser ms intuitivas y rpidas para ti. Primero, instala y carga el paquete:

library(dplyr)

Subconjunto por nmeros de fila

Si conoces los nmeros de fila en los que ests interesado en hacer el subconjunto, puedes seleccionar fcilmente estas filas usando corchetes como se discuti anteriormente o la funcin slice en dplyr.

Por ejemplo, para seleccionar solo las filas 10-12 en el marco de datos, usaras:

Bats.slice <- slice(Bats, 10:12)

Seleccionar filas que cumplan ciertos criterios

Podemos hacer un subconjunto de estas filas usando la funcin filter. Por ejemplo, si solo queremos las filas donde se registr actividad de forrajeo, podramos seleccionar las filas donde el recuento de llamadas de forrajeo es mayor que cero.

Bats.foraging <- filter(Bats, Foraging > 0)

Tomar una seleccin aleatoria de filas

Es posible que deseemos seleccionar aleatoriamente un nmero o fraccin de filas en nuestro conjunto de datos para validar nuestros modelos. Por ejemplo, podramos tomar el 50% de los datos para construir el modelo y luego el 50% del modelo para probarlo con datos observados. La seleccin aleatoria de filas es importante si vamos a dividir los datos de esta manera, porque no queremos sesgar nuestros datos hacia ninguna propiedad o categora en particular. Aqu creamos un nuevo marco de datos Bats.50p que tiene la mitad del nmero de filas del conjunto de datos inicial:

Bats.50p <- sample_frac(Bats, size = 0.5, replace = FALSE)

De manera similar, podramos solicitar que se muestre un cierto nmero de filas seleccionadas al azar del conjunto de datos. Por ejemplo, para seleccionar al azar 100 filas, usaramos:

Bats.100r <- sample_n(Bats, 100, replace = FALSE)

Eliminar filas duplicadas

Otra funcin til es eliminar filas duplicadas, por ejemplo, si hemos ingresado datos dos veces por accidente.

Podemos comparar el nmero de filas entre nuestro conjunto de datos original y nuestro nuevo conjunto de datos sin duplicados y nuestro conjunto de datos anterior utilizando <nrow:

En este caso, los dataframes son idnticos porque nuestros datos no tienen filas duplicadas.

Seleccin de columnas en dplyr

Como se mencion anteriormente, puedes seleccionar una columna $ o una o ms mediante indexacin con []. dplyr tiene la funcin select que te permite seleccionar columnas por nombre o mediante el uso de funciones auxiliares tiles.

Seleccionar columnas por nombre

Por ejemplo, para seleccionar solo la columna Site del marco de datos, o tanto las columnas Site como Date:

Bats_subset1 <- select(Bats, Site)
Bats_subset2 <- select(Bats, Site, Date)

Seleccionar varias columnas por su posicin

Para seleccionar un grupo de columnas adyacentes entre s, utiliza los nombres de la primera y ltima columna separados por :. Por ejemplo, esto selecciona todas las columnas entre Site y Date.

Bats_subset3 <- select(Bats, Site:Date)

Seleccionar columnas por sus propiedades

Las funciones auxiliares incorporadas en select te permiten seleccionar columnas particulares segn sus propiedades.

Por ejemplo, podramos seleccionar las dos columnas (Distance.murray.water y Distance.creek.water) por el inicio de los nombres de columna o por el final de los nombres de columna:

Bats_subset4 <- select(Bats, starts_with("Distance"))
Bats_subset4 <- select(Bats, ends_with("water"))

Si ests familiarizado con las expresiones regulares, puedes usar la funcin auxiliar match. Por ejemplo, para seleccionar esas mismas dos columnas, podramos usar ^ para indicar que el nombre de la columna comienza con los siguientes caracteres y $ para indicar que el nombre de la columna termina con los caracteres precedentes.

Bats_subset4 <- select(bats, matches("^Distance"))
Bats_subset4 <- select(bats, matches("water$"))

Podemos usar contains cuando queremos seleccionar columnas que contienen ciertos caracteres o palabras en su nombre.

Bats_subset4 <- select(Bats, contains("water"))

La funcin auxiliar one_of seleccionar columnas que pertenezcan a una lista de nombres de columnas, recordando usar c() para concatenar la lista de nombres.

Bats_subset4 <- select(bats, one_of(c("Distance.murray.water", "Distance.creek.water")))

Ten en cuenta que los ltimos 6 usos de select hicieron exactamente lo mismo: extrajeron las dos columnas, Distance.murray.water y Distance.creek.water.

Tambin puedes nombrar columnas especficas que deseas excluir de la seleccin usando un signo menos delante del nombre de la columna a excluir. Por ejemplo, para crear un marco de datos que ya no tenga la variable Foraging:

Bats_subset5 <- select(Bats, -Foragaing)

Por ltimo, si tienes columnas numeradas, podemos usar la funcin auxiliar numrange para seleccionar columnas especficas. Por ejemplo, este cdigo seleccionara las columnas llamadas var1, var2 y var3.

select(data, num_range("var", 1:3))
