Algunas pruebas de hipótesis simples con la estadística t:

  • Pruebas t de una muestra para contrastar un parámetro de muestra con un parámetro de población
  • Pruebas t de muestras independientes para contrastar las medias de dos muestras
  • Pruebas t emparejadas para contrastar dos grupos de valores donde cada par se mide en el mismo objeto.
