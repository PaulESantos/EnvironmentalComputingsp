Creacin de un grafico con ggplot

Primero, necesitars instalar el paquete ggplot2 en tu mquina y luego cargar el paquete con la funcin library como de costumbre.

library(ggplot2)

El punto de partida para crear un grfico es utilizar la funcin ggplot con la siguiente estructura bsica:

plot1 <- ggplot(data, aes(x_variable, y_variable)) +
  geom_graph.type()

print(plot1)

data es un data frame que contiene las variables que deseas representar en el grfico.

aes especifica qu variables se deben representar. Este cdigo suele causar confusin al crear un objeto ggplot. Aunque aes significa esttica, en ggplot no se refiere al aspecto visual del grfico, sino a los datos que deseas ver en el grfico. Especifica qu se presenta en el grfico en lugar de cmo se presenta.

+ geom_ especifica qu tipo de grfico quieres crear, ggplot no funcionar a menos que agregues esto.

Algunos ejemplos comnmente utilizados incluyen: * scatterplot (grfico de dispersin) - + geom_point() * boxplot (grfico de caja) - + geom_boxplot() * histograma - + geom_histogram() * grfico de barras - + geom_bar()

Asegrate de tener un parntesis abierto y cerrado despus del cdigo geom_. Esto le indica a R que haga el grfico con el formato estndar bsico para este tipo de grfico.

El tipo de grfico que deseas hacer debe coincidir con las clases de las variables de entrada. Por ejemplo, un scatterplot requerira que ambas variables sean numricas. O un boxplot requerira que la variable x sea un factor y la variable y sea numrica. Puedes verificar la clase de cualquier variable con la funcin class, o todas las variables en un data frame con la funcin str.

Una prctica til al crear objeto con ggplot es asignar el grfico que has creado a un objeto (por ejemplo, plot1 en el cdigo anterior) y luego imprimirlo por separado. A medida que tu ggplot se vuelve ms complicado, esto facilitar mucho las cosas.

Ejemplos de grficos

En estos ejemplos, utilizaremos un conjunto de datos que ya est en R con la longitud y el ancho de partes florales de tres especies de iris. Primero, carga el conjunto de datos:

data(iris)

Para contrastar las longitudes del ptalo entre las tres especies de iris con un diagrama de caja, usaramos:

plot1 <- ggplot(iris, aes(Species, Petal.Length)) +
  geom_boxplot()
print(plot1)

Observa cmo aes incluye las variables x y luego y que deseas graficar, y cmo + geom_boxplot() especifica un diagrama de caja.

Para graficar un histograma de frecuencia de la longitud del ptalo, usaramos:

plot2 <- ggplot(iris, aes(Petal.Length)) +
  geom_histogram()
print(plot2)

Para generar un grfico de dispersin y visualizar la relacin entre la longitud del ptalo y la longitud del spalo, usaramos:

plot3 <- ggplot(iris, aes(Sepal.Length, Petal.Length)) +
  geom_point()
print(plot3)

Estos puntos podran ser coloreados segn los niveles de una variable categrica. Para hacer esto, agrega colour="variable.categrica" dentro de los parntesis de aes. Para colorear por especie, usaramos:

plot4 <- ggplot(iris, aes(Sepal.Length, Petal.Length, colour = Species)) +
  geom_point()
print(plot4)

Aadiendo un tema bsico

La apariencia general del ggplot se puede modificar con diferentes temas predefinidos. Esto se puede hacer agregando + theme_bw() o + theme_classic() al final de tu lnea de cdigo. Al igual que con geom(), asegrate de que haya parntesis abiertos y cerrados despus del nombre del tema.

plot5 <- ggplot(iris, aes(Sepal.Length, Petal.Length, colour = Species)) +
  geom_point() +
  theme_bw()
print(plot5)

plot6 <- ggplot(iris, aes(Sepal.Length, Petal.Length, colour = Species)) +
  geom_point() +
  theme_classic()
print(plot6)
