Diagramas de Caja

Los diagramas de caja se realizan f√°cilmente con la funci√≥n boxplot. Los diagramas de caja muestran la distribuci√≥n de una variable indicando la mediana, los cuartiles, el m√°ximo y el m√≠nimo de una variable. La l√≠nea gruesa negra es la mediana, y las cajas a cada lado de la l√≠nea de mediana son los cuartiles inferiores y superiores.

Para contrastar la tasa metab√≥lica entre las dos especies, utilizar√≠amos:

La variable continua se encuentra a la izquierda del s√≠mbolo de tilde (~) y la variable categ√≥rica se encuentra a la derecha. De inmediato, puedes observar que la especie B tiene una tasa metab√≥lica m√°s alta que la especie A.

Gr√°ficos de barras

Este tipo de datos tambi√©n se visualiza com√∫nmente mediante un gr√°fico de barras que muestra las medias de la variable continua para cada nivel de la variable categ√≥rica, con barras de error que muestran algunas medidas de variaci√≥n dentro de cada categor√≠a. Las barras de error pueden ser la desviaci√≥n est√°ndar, el error est√°ndar o los intervalos de confianza del 95%.

Aunque se utilizan com√∫nmente, no son tan f√°ciles de crear con las funciones b√°sicas de R. Hay varias formas de hacerlo, pero una opci√≥n es utilizar las funciones summarise y group_by del paquete dplyr para calcular las medias y medidas de variaci√≥n para cada nivel de tu variable categ√≥rica (consulta Resumen de datos).

Aqu√≠ tienes un c√≥digo de ejemplo para visualizar las medias +/- desviaciones est√°ndar. Para tener un mayor control sobre las barras de error, te recomendamos utilizar opciones de trazado m√°s avanzadas en el paquete ggplot2 (consulta gr√°ficos de barras con barras de error).

library(dplyr)

Species.summary <- Prawns %>% # the names of the new data frame and the data frame to be summarised
  group_by(Species) %>% # the grouping variable
  summarise(
    mean = mean(Metabolic_rate), # calculates the mean
    sd = sd(Metabolic_rate), # calculates the standard deviation
    lower = mean(Metabolic_rate) - sd(Metabolic_rate),
    upper = mean(Metabolic_rate) + sd(Metabolic_rate)
  )

En un nuevo data frame llamado Species.summary, ahora tenemos las medias, desviaciones est√°ndar y los valores inferiores y superiores que establecen el tama√±o de las barras de error para cada nivel de la variable de agrupaci√≥n. Los l√≠mites de las barras de error se calcularon sumando (superior) o restando (inferior) la desviaci√≥n est√°ndar a la media.

Ahora podemos utilizar las funciones barplot y arrows para crear un gr√°fico con barras de error.

Prawn.plot <- barplot(Species.summary$mean,
  names.arg = Species.summary$Species,
  ylab = "Metabolic rate", xlab = "Species", ylim = c(0, 1)
)

Ten en cuenta que los gr√°ficos de medias y barras de error pueden ser enga√±osos, ya que ocultan las distribuciones reales de los datos. Las medias tambi√©n pueden ser enga√±osas cuando los datos est√°n muy sesgados y los c√°lculos para las barras de error utilizando estad√≠sticas t (por ejemplo, intervalos de confianza del 95%) asumen que los datos est√°n distribuidos de manera normal.
